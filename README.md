# Akash Dairy and Grocery – Sales Analysis Dashboard

## Project Overview
This project involves designing an **interactive Power BI dashboard** for Akash Dairy and Grocery to analyze sales performance, product distribution, outlet trends, and customer preferences. The dashboard enables data-driven decision-making through clear KPIs and visual insights.

---

## Objectives
- Analyze total and average sales performance  
- Identify top-performing product categories  
- Compare sales by outlet size, type, and fat content  
- Track outlet establishment trends over time  

---

## Tools & Technologies
- Power BI  
- Microsoft Excel  
- DAX (basic measures and KPIs)  

---

## Key Features
- KPI cards for Total Sales, Average Sales, Number of Items, and Ratings  
- Sales trend analysis by outlet establishment year  
- Category-wise sales comparison  
- Outlet size and fat content analysis  
- Interactive filters for dynamic exploration  

---

## Dataset
- Grocery sales dataset (Excel format)  
- Cleaned and transformed using Power BI  

---

## Files Included
- `Akash Diary and Grocery.pbix` – Power BI dashboard file  
- `Akash Diary and Grocery.pdf` – Exported dashboard report  
- `BlinkIT Grocery Data.xlsx` – Source dataset  
- `UPTRIX INTERNSHIP PROJECT REPORT.docx` – Detailed project report  

---

## Outcome
The dashboard provides **clear business insights** into sales distribution and customer demand patterns, helping stakeholders optimize inventory planning and outlet performance.
